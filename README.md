# Smart Interview Assistant

A multi-modal interview coaching tool that analyzes uploaded audio/video (or live webcam)
and returns feedback on fluency, filler words, answer structure, nonverbal cues, and more.

This repository contains:
- a FastAPI backend for processing uploads
- a Streamlit frontend for interactive demos
- processing modules: STT (WhisperX), audio features, filler detection, NLP scoring, and video/CV analysis

See `docs/` for rubric and evaluation details.
